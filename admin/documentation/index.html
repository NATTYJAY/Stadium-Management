Getting started

Hyrax UX is fully responsive bootstrap admin template for your complete need.

Features

Bootstrap Framework

Bootstrap easily and efficiently scales your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.

Easily customizable

Responsive layout

Bitcoin Price Chart Live Widgets

For Dark Themed chart, use the below Code and place at your desired location

<div class="btcwdgt-chart"></div>

For Light Themed chart, use the below Code and place at your desired location

<div class="btcwdgt-chart" bw-theme="light"></div>


Place the following script at the bottom of page

<script src="plugins/assets/js/stock-chart.init.js"></script>
<script src="plugins/assets/js/bitcoin-init.js"></script>
